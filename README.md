# solana-web3js-v3-rc-1-test

**Written for Solana Devnet**

Run it to console log teh address/public keys if teh wallets involved & teh signature of teh transfer transaction between them. Copy teh transaction and check it on your favoured explorer.

## Setup

```bash
git clone https://github.com/rohaan-sahu/solana-web3js-v3-rc-1-test.git
cd solana-web3js-v3-rc-1-test
pnpm install
```

Alternatively, use `npm install` or `yarn install`. 

## Run

```bash
npx tsx web3jsv3demo.ts
```
